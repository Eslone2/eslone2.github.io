Windowsとのおもひで

Posted on November 27, 2025 by Eslone | 雑記

はじめに

こんにちは、えすろんです。今回は、過去のWindowsとの思い出を紹介したいと思います。

Windows XP

リソースを食わないためとてもサクサクだった記憶がある。
最初のマシンであるPentium IIIでも案外サクサクだった記憶がある。
・RAM: DDR4 2400 16GBx4 (合計64GB)
・GPU: intel UHD Graphics 630 (内蔵グラフィックス)
・ストレージ: NVMe SSD 500GB + SATA SSD 480GBx2 + HDD 1TB
・OS: Windows 11 Pro 64bit
・モニター: 22インチ 1920x1080 60Hz
このPCでは、プログラミング(VSCode上でのPython, C++等)や動画編集がメインになっています。
性能が不足気味になってきたのですが、いかんせん予算が厳しいので、しばらくはこのまま使い続ける予定です。

メイン機(2号)

メイン機(2号)はノートパソコンで、以下のスペックです。
・機種: SHARP Dynabook V8 P1V8WPBL
・CPU: Intel Core i7 1360P
・RAM: LPDDR4 16GB
・GPU: intel Xe Graphics 96EU (内蔵グラフィックス)
・ストレージ: NVMe SSD 500GB
・OS: Windows 11 Pro 64bit
・モニター: 13.3インチ 1920x1080 60Hz タッチ対応
このPCでは、プログラミング(VSCode上でのPython実行)やWeb開発を行っています。
また、外出先での作業にも使用しています。意外とタッチ機能が便利です。

サブ機

サブ機はノートPCで、以下のスペックです。
・機種: NEC Lavie Direct NM PC-GN12S78AA
・CPU: Intel Core i5 7Y54
・RAM: LPDDR3 8GB
・GPU: intel HD Graphics 615 (内蔵グラフィックス)
・ストレージ: SATA SSD 256GB
・OS: Windows 11 Home 64bit(魔改造済み)
・モニター: 11.6インチ 1920x1080 60Hz
こっちでは主にColaboratoryを使ったPythonの実行やドキュメント作成を行っています。

サブ機(2号)

サブ機(2号)はタブレットPCで、以下のスペックです。
・機種: GZBOX Surface NUC 1
・CPU: Intel Processor N150
・RAM: LPDDR5 16GB
・GPU: intel UHD Graphics (内蔵グラフィックス)
・ストレージ: SATA SSD 512GB
・OS: Windows 11 Pro 64bit
・モニター: 7インチ 1920x1080 60Hz タッチ対応
この機種は主に外出先での軽作業用として使用しています。

簡易サーバ

簡易サーバとしてDELL inspiron 3470を使用しています。以下のスペックです。
・CPU: Intel Core i3 8100T
・RAM: DDR4 2400 8GBx2 (合計16GB)
・GPU: intel UHD Graphics 630 (内蔵グラフィックス)
・ストレージ: SATA SSD 480GB
・OS: Windows 11 Home 64bit
このPCは、実家でファイル共有や簡易的なDLNAサーバとして利用しています。

過去のPCたち

IBM ThinkPad X24(2010~2016)
・CPU: Intel Pentium III 1.0GHz
・RAM: 256MB
・GPU: Intel Extreme Graphics 2 (内蔵グラフィックス)
・ストレージ: HDD 20GB
・OS: Windows XP Professional 32bit
このPCは、初めて使ったPCで、主にネットサーフィンに使用していました。
液晶が御臨終してしまい、廃棄しました。

HP mini 5103(2010~2017)
・CPU: Intel Atom N550
・RAM: DDR3 2GB
・GPU: intel GMA 3150 (内蔵グラフィックス)
・ストレージ: HDD 160GB -> 後にSSD 120GBに換装
・OS: Windows 7 Home Premium 32bit
このPCはサブ機でしたが、動作が非常に遅いため、あまり使いませんでした。

Lenovo ThinkPad SL500(2011~2021)
・CPU: Intel Core 2 Duo T9600
・RAM: 4GB
・GPU: intel GMA 4500MHD (内蔵グラフィックス)
・ストレージ: HDD 250GB(後にSSDに換装)
・OS: Windows 7 Professional 64bit -> Windows 8.1 Pro 64bit -> Windows 10 Pro 64bit
このPCは、高校生になる直前まで使用していました。
このマシン、意外と動作が軽快でした。

DELL Inspiron 660S(2014~2022)
・CPU: Intel Core i5 3330S
・RAM: DDR4 8GBx2 (合計16GB)
・GPU: intel HD Graphics 2500 (内蔵グラフィックス)
・ストレージ: HDD 1TB -> 後にSSD 240GBに換装
・OS: Windows 7 Home Premium 64bit -> Windows 10 Pro 64bit
このPCは、小学校入学後からメイン機として使用していました。

自作PC(2021~2024)
・CPU: intel Core i5 11400
・MB: ASRock Z590 Phantom Gaming itx/TB4
・RAM: DDR4 2400 16GBx2 (合計32GB)
・GPU: intel UHD Graphics 730 (内蔵グラフィックス) -> 後にGTX 1060(6GB)を追加
・ストレージ: NVMe SSD 500GB + SATA SSD 480GB
・OS: Windows 10 Pro 64bit -> Windows 11 Pro 64bit
このPCは、高校生になってから大学入学まで使用していました。
CPUが故障するという珍しいトラブルがあり、その後GPUも故障したため、現在のメイン機に買い替えました。
東方Projectのプレイ中にBSoDが頻発して大変だった記憶があります。
