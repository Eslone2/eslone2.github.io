💻 ネットワーク構築 講習会（第二回）

Posted on May 6, 2026 by Eslone | 講習会資料

🎯 今回の最終目標

  • Teratermを使ってネットワーク機器（ルータやスイッチ）に接続できる。
  • 機器の基本的な操作モードの違いを理解し、状態確認コマンドを実行できる。

📥 1. Teratermでの接続方法

第1回でインストールした「Teraterm」を使用して、ネットワーク機器（Ciscoルータやスイッチ）に接続します。

多くの場合、初期設定を行う際は「コンソールケーブル」と呼ばれる専用のケーブルを使ってPCと機器を直接繋ぎます（シリアル接続）。

  1. PCとネットワーク機器をコンソールケーブルで接続します。
  2. Teratermを起動し、「シリアル(Serial)」を選択します。
  3. ポート(Port)を選択し、「OK」をクリックします。
注意： シリアルポートの設定（設定 -> シリアルポート）で、スピード(Baud rate)が 9600 になっていることを確認してください（Cisco機器の標準設定）。

⚙️ 2. ネットワーク機器の3つのモード

無事に接続できると、黒い画面に文字が表示されます。Cisco機器（IOS）には、操作の権限によって主に3つのモードが存在します。

① ユーザーEXECモード ( > )

接続直後のモードです。プロンプト（入力待ちの記号）は Router> のようになります。

このモードでは、限られた確認コマンドしか実行できず、設定の変更などは一切できません。

② 特権EXECモード ( # )

すべての確認コマンドや、ファイルの保存・再起動などができるモードです。プロンプトは Router# に変わります。

ユーザーEXECモードから移行するには、以下のコマンドを入力します。

Router> enable
Router#

③ グローバルコンフィギュレーションモード ( (config)# )

機器全体の設定（IPアドレスの付与やルーティングの設定など）を行うモードです。プロンプトは Router(config)# に変わります。

Router# configure terminal
Router(config)#

元のモードに戻るには exit コマンドを使用します。

🔍 3. 状態確認の基本コマンド

特権EXECモード（#）でよく使う、非常に重要な確認コマンドを2つ紹介します。

show ip interface brief

各インターフェース（LANケーブルを挿すポート）のIPアドレスや、通信できる状態（UP/DOWN）を一覧で確認できます。

Router# show ip interface brief
Interface IP-Address OK? Method Status Protocol
GigabitEthernet0/0 192.168.1.1 YES manual up up
GigabitEthernet0/1 unassigned YES unset administratively down down

show running-config

現在機器で動いている「すべての設定（コンフィグ）」を表示します。自分が設定した内容が正しく反映されているかを確認するために使います。

Router# show running-config
Building configuration...
(以下、設定内容がずらっと表示されます。スペースキーでページ送りできます)
