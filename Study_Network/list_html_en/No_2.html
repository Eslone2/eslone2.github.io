💻 Network Construction Workshop (Session 2)

Posted on May 6, 2026 by Eslone2

🎯 Final Goals for this Session

  • Be able to connect to network devices (routers and switches) using Teraterm.
  • Understand the differences between basic operating modes and execute status check commands.

📥 1. How to Connect with Teraterm

Use "Teraterm" installed in Session 1 to connect to network devices (Cisco routers and switches).

In most cases, when performing initial configuration, you connect the PC directly to the device using a special cable called a "console cable" (serial connection).

  1. Connect the PC and the network device with a console cable.
  2. Launch Teraterm and select "Serial".
  3. Select the Port and click "OK".
Note: Ensure that the speed (Baud rate) is set to 9600 in the serial port settings (Setup -> Serial port) (the standard setting for Cisco devices).

⚙️ 2. Three Modes of Network Devices

Once successfully connected, text will appear on a black screen. Cisco devices (IOS) mainly have three modes depending on operating privileges.

① User EXEC Mode ( > )

This is the mode immediately after connecting. The prompt (input waiting symbol) looks like Router>.

In this mode, you can only execute limited verification commands, and no configuration changes can be made.

② Privileged EXEC Mode ( # )

In this mode, you can execute all verification commands, save files, restart, etc. The prompt changes to Router#.

To transition from User EXEC mode, enter the following command.

Router> enable
Router#

③ Global Configuration Mode ( (config)# )

This is the mode for configuring the entire device (such as assigning IP addresses and setting up routing). The prompt changes to Router(config)#.

Router# configure terminal
Router(config)#

Use the exit command to return to the previous mode.

🔍 3. Basic Status Check Commands

Here are two very important verification commands frequently used in Privileged EXEC Mode (#).

show ip interface brief

You can view a list of IP addresses for each interface (ports where LAN cables are plugged in) and their communication status (UP/DOWN).

Router# show ip interface brief
Interface IP-Address OK? Method Status Protocol
GigabitEthernet0/0 192.168.1.1 YES manual up up
GigabitEthernet0/1 unassigned YES unset administratively down down

show running-config

Displays "all configurations (configs)" currently running on the device. Used to verify if the settings you made have been correctly applied.

Router# show running-config
Building configuration...
(Below, the settings will be displayed in a long list. You can press the spacebar to page down)
