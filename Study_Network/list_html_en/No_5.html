💻 Network Construction Workshop (Session 5)

Posted on May 12, 2026 by Eslone2

🎯 Final Goals for this Session

  • Understand what VLAN (Virtual LAN) is.
  • Understand the mechanism of VLAN tags (IEEE 802.1Q).
  • Understand the difference between Access Ports and Trunk Ports.

🌐 What is VLAN (Virtual LAN)?

VLAN is a technology that logically divides a network (broadcast domain) within a switch, regardless of physical connections.

Benefits of Using VLANs

  • Improved Security: You can isolate networks by department or purpose, blocking unnecessary communication.
  • Network Efficiency: By reducing the scope of broadcast traffic (broadcast domain), you can decrease unnecessary traffic.
  • Flexible Configuration: Even if physical locations change, devices can remain in the same network through configuration changes.

🏷️ VLAN Tags and IEEE 802.1Q

When configuring VLANs across multiple switches, it's necessary to identify which traffic belongs to which VLAN. The identifier used for this is the **VLAN Tag**.

The most common standard today is **IEEE 802.1Q**.

💡 How VLAN Tags Work

A "tag" containing the VLAN number (VLAN ID) is inserted into a standard Ethernet frame. This allows multiple VLANs' data to be carried over a single cable.

🔌 Port Types: Access and Trunk

Switch ports primarily have two roles:

Port Type Characteristics Main Use
Access Port Belongs to only one VLAN. No tag is added. Connecting end devices like PCs or printers.
Trunk Port Carries traffic from multiple VLANs over one link. Identified by tags. Connecting switches together or to a router.

⚙️ Configuration Example (Cisco)

These are the basic commands for configuring VLANs on a Cisco switch (e.g., Catalyst).

1. Creating a VLAN

Switch(config)# vlan 10
Switch(config-vlan)# name Sales

2. Configuring an Access Port

Assign a port to a specific VLAN.

Switch(config)# interface gigabitEthernet 0/1
Switch(config-if)# switchport mode access
Switch(config-if)# switchport access vlan 10

3. Configuring a Trunk Port

Configure a port to carry traffic from multiple VLANs.

Switch(config)# interface gigabitEthernet 0/24
Switch(config-if)# switchport mode trunk

📝 Summary

  • VLAN: Technology to logically divide networks.
  • VLAN Tag: A label used to identify VLANs between switches.
  • Access Port: For a single VLAN (terminal-facing).
  • Trunk Port: For multiple VLANs (switch/router-facing).
